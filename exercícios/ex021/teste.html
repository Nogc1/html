Grouping Tags

Grouping Tags são tags agrupadas para uma melhor funcionalidade do site e uma melhor visualização do código.

As grouping tags são separadas em dois estilos de caixas inline-level e block-level.

    Block-level

  • div
  • h1 - h6
  • p
  • main
  • header
  • nav
  • article
  • aside
  • footer
  • form
  • video

    Inline-level

  • span
  • a
  • code
  • small
  • strong
  • em
  • sup - sub
  • label
  • button
  • input
  • select
